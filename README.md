# SQL-Ecommerce-Analysis
# SQL E-commerce Analysis Project

## Project Overview
This is a beginner SQL project based on an e-commerce dataset.  
The project analyzes customers, products, and orders using Oracle SQL.

## Dataset
The project contains three tables:

1. Customers
- customer_id
- gender
- age
- city
- signup_date
- loyalty_member

2. Products
- product_id
- product_name
- category
- price

3. Orders
- order_id
- customer_id
- product_id
- order_date
- quantity
- payment_method

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate functions
- INNER JOIN
- LEFT JOIN
- Filtering
- Revenue calculation

## Analysis Performed
- Total customers
- Total orders
- Total products
- Customers by city
- Products by category
- Most used payment method
- Top sold products
- Revenue by product
- Revenue by payment method
- Customers who never ordered
- Products that were never ordered

- Oracle SQL Developer
- Oracle Database
